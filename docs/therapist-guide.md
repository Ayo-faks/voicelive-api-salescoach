# SpeakBright Therapist Quick Start

SpeakBright is a therapist-supervised speech practice tool for short child sessions. Practice feedback supports the session and is not a clinical assessment.

## Before you begin

- Complete the short therapist onboarding flow the first time you open the app in a browser.
- Confirm the therapist PIN when prompted so review tools stay in therapist hands.
- Read and acknowledge the supervised-practice consent before the first child session.

## Create or select exercises

- Open the main practice screen.
- Choose a built-in exercise from the exercise cards, or create a therapist exercise in the "Therapist exercises" section.
- For custom exercises, enter the target sound, target words, difficulty, and child-facing prompt.

## Start a child session

- Select the child profile at the top of the practice screen.
- Select the exercise you want to use.
- Tap "Start child session" and hand the device to the child.
- Stay nearby during the session and guide the child as needed.

## Review progress

- After the session, open the results dialog for the immediate practice summary.
- Use "Open therapist review" to see saved sessions for each child.
- In therapist review, open a saved session to inspect articulation, engagement, pronunciation, and any therapist feedback already recorded.

## Submit post-session feedback

- In the results dialog, choose either "Helpful session" or "Needs follow-up".
- Add an optional short note if something from the session should be remembered.
- Save the feedback so it appears with the stored session record.

## Known limitations

- SpeakBright is for supervised practice only and is not a diagnostic tool.
- Telemetry is limited to pilot operations signals and excludes child names, transcripts, and speech content.
- Mobile browsers are supported for pilot use, but the best experience is still a stable connection with a therapist nearby.
