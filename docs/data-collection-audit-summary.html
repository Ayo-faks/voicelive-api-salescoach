What is Wulo?

Wulo is a web-based speech therapy practice tool for children with speech and language difficulties. A therapist (or parent) selects exercises, the child speaks into the microphone, and AI provides pronunciation feedback. The app also generates personalised practice plans and tracks progress over time.

What personal data do we collect?

About therapists / parents (the adult users)

Is any of this sensitive data?

Yes. Speech transcripts, pronunciation assessments, clinical notes, and AI observations about a child’s speech development likely constitute health data under UK GDPR Article 9. This is because the children are using the app specifically because they have speech/language difficulties.

Lawful basis: Explicit parental consent, recorded in the app before any child data is processed.

Where does the data go?

All data is processed and stored within Microsoft Azure, located in Sweden Central (EU). We do not use Google, Anthropic, or any non-Microsoft AI service.

Data flow for a typical session

Child speaks → Browser captures audio
    → Azure Speech Service (Sweden Central)
        Converts speech to text, scores pronunciation
    → Azure OpenAI (Sweden Central)
        Evaluates the child's speech against the exercise target
        Generates feedback and progress observations
    → Azure PostgreSQL (Sweden Central)
        Stores transcript, scores, and AI assessment

What we do NOT use

  • Google Cloud / Vertex AI — not used
  • Anthropic / Claude — not used
  • OpenAI (direct) — not used; we use Azure OpenAI, which is hosted and governed by Microsoft under their Azure data processing terms
  • AWS — not used

How long do we keep data?

What How long Then what
Active child profiles As long as the child is using the app —
Inactive children (no session for 6 months) Automatically flagged for deletion Soft-deleted, then hard-deleted after 1 month grace period
Consent records Kept even after withdrawal Required to prove consent was obtained
Audit log Kept indefinitely Required for GDPR accountability

What rights do users have?

Right How it works
Access (Art. 15) Therapist can export all data for a child as a JSON file
Erasure (Art. 17) Therapist can permanently delete a child and all associated data
Withdraw consent Parent/guardian can withdraw consent; processing stops
Contact privacy@wulo.ai

Cookies

Only one cookie is set by default: the consent preference cookie (strictly necessary).

Analytics cookies (Microsoft Clarity) are only set after the user explicitly opts in via a cookie banner. They can be revoked at any time.

No advertising or tracking cookies are used.
